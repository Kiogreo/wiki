# Kiogreo Wiki

## About

A Vitepress static wiki website for Kiogreo Ecosystem

## Quickstart

### Local Development

1. Run the below command to start hosting the wiki in your localhost:

    ```bash
    pnpm install && pnpm dev
    ```

2. From your web browser, access the wiki through `http://localhost:5173` (default local hosting address)

Refer to Github Workflow script at `.github/workflows/build-and-deploy.yaml`
